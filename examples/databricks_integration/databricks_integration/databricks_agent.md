# Databricks agent

## Installation

The Databricks agent comes bundled with the Spark plugin. To use the Spark plugin, install it with `pip`:

```
pip install flytekitplugins-spark

```

## Example usage

For a usage example, see the {doc}`Databricks job <databricks_job>` page.
